Create the director script.

Copy a ready-made LLM brief with the song timing, video type, cast, and required format. Paste the returned director script here when it is ready.

1. Copy LLM brief
The app builds this prompt from Step 1-3, including the SRT timing, selected video type, cast names, reference notes, and the exact script format.
SRT timing included
The user does not need to find or copy timing manually. The generated brief includes the timed lyric lines from Step 1.
2. Paste director script
The returned script becomes the plan. ComfyStudio parses the timing, shot type, keyframe prompt, and motion prompt from this box.
Format guardrails
The parser expects each shot to include Start at, Shot type, Keyframe prompt, Motion prompt, and Length. Artist and Lyric moment are optional but recommended for lip-sync shots.
Timing must be ready first
If lyric timing is still preparing in Step 1, the LLM brief should wait. Better timing here means fewer cropped or mistimed performance shots later.

What this Easy Mode is protecting the user from
They do not need to understand SRT, shot type grammar, pass signatures, stale plans, or workflow ids before they can create. Those remain available in advanced controls.
